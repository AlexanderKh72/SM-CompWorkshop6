Подробное описание

Дискретное распределение.

Класс Distribution хранит произвольное дискретное распределение с конечным числом состояний. На вход (в конструкторе или функции Distribution::set) подаётся массив с весами состояний и число состояний. При помощи функции Distribution::get_p(n) можно получить вероятности с котороми случайная величина принимает заданное значение.
